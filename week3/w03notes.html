Chapter 6

  • Use typeof X to get the data type of X
  • document.body returns the body element
  • document.images retures a list of images in the document
  • document.links returns a list of links
  • document.getElementsByClassName('blah') will get element by class
  • To add styles to the elements through javascript follow the example
    const heroes = document.getElementById('roster');
    const superman = heroes.children[0];
    superman.style.border = "red 2px solid";
    << "red 2px solid"

Chapter 7

  • preventDefault() will stop a default befavior from happening.

    const brokenLink = document.getElementById('broken');
    brokenLink.addEventListener('click',(event) => {
    event.preventDefault();
    console.log('Broken Link!');
    });

    this will stop the link from working normally and display in the console that it is broken.
  • stopPropagation() will stop events higher up in the chain. So be careful when using it.
