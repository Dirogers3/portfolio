W7 Notes

Chapter 11

  • You can use .length on a function to get the number of parameters the function takes.
  • You can use .apply to call a function with an array of parameters.
  • You can use .call to call a function with a list of parameters.
  • You can use .bind to call a function with a list of parameters and a context.
  • Don't use callbacks a lot, Make sure code is simple, clean and easy to read.
  • Higher order functions are functions that return other functions.
  • To use a generator put a '*' after the function word.

Chapter 13 AJAX

  • AJAX allows web pages to communicate asynchronously with a server.
  • You can fetch API by using fetch(url).then(do more stuff)
  • JSON is the most common format for AJAX
  • Use Get to retrieve information
  • Use POST to put information in the database.
  • const data = new FormData(); this makes it easy to submit form using AJAX
