Chapter 9: The Window Object

  • window.confirm('Do you wish to continue?') will prompt the user with a cancel and okay buttons that return a boolean.
  • window.propt('please enter your name:') will prompt an input box for the user to type something in. Text will be return as a string if clicked on OK
  • window.navigator.userAgent will tell you the browser and OS
  • window.history.go(1) to go forward 1 page, (-1) to go back one page.
  • To determine the screen width of the device you can do window.screen.width
  • cookies are local storage on the browser for the user. We shouldn't store any sensative data on this though as anyone in can get into this.
