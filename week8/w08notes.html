Chapter 8: Transform and Transitions

  • Transform property can change the html element's position, size, and rotation.
  • Transition property can change the html element's properties over a period of time.
  • You can add the ransform in a element tag and put it within the element attributes
  • Example:
    .square {
    background: darkturquoise;
    border-radius: 5px;
    height: 100px;
    margin: 100px;
    transition: transform 1s; width: 100px;

    &:hover {
    transform: scale(2);
    }border-radius }
