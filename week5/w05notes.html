W5 Notes

  • Strice mode will allow more exceptions and warnings and doesn't allow the use of deprecated features. With this mode on, you can really fine tune and perfect your code. To use this just type 'use strict'; on the first line of the file.
  • The browser is a good place to debug. Use the console and inspect element tools to check your functions and variables.
  • You an also use the alert(); function to debug as well. This shows a dialog and allows us to create breakpoints in the code.
  • When shipping the application, remvoe all the debugging code.
  • use try and catch if a suspected error might happen.
