Notes from the video

Chapter 2

  • You can do @imports url (....) to import fonts from google right into your CSS. I've always just added the link tag in the html.
  • Primative data types are string, symbol, number, boolean, undefined, and null. Everything else is an object (including arrays) You can use the typeof method to see what the data type is.
  • concat method will combine two things side by side.
  • Template Literals use the ` back tick marks and we can use variables too. `Hello ${name}!`; this will return Hello, Ian
  • Number.toFixed('number of digits after the decimal')
  • Exponents are double * so it would look like 2**3 = 8
  • Number('23') will change to an actual number of 23
  • Null behaves like a zero and undefined is actual NaN

Chapter 3

  • Pop will remove the last item from an array
  • Push will put something in at the end of the array
  • shift will remove the first item in the array
  • Unshift will add something in at the beginning of the array
  • concat can be used to merge arrays
  • join() can be used to turn an array into a string and if join(' & ') is used it will put a & between each item in the array.
  • Slice(a,b) creates a subarray from index a to index b
  • splice(a,b,'stuff'): a is the index to start the remove and b is the number of items from a. and stuff will be replaced for that spot. (will return removed item)
  • array.sort() is alphabetically
  • .add() for sets( {a, b, c, d} )
  • spread operator ... set to array const array = [...set]
  • Ternary Operator: condition ? (//code to run if condition is true) : (//code to run if condition is false)
  • switch(x { case:x blah; break;})

Chapter 4

  • Default paramater just need to have the variable equal something in the parameter ()
  • Hoisting: variables and functions can be used if they are defined later in the code.
