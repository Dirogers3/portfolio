Chapter 8 Forms

  • action='/method' to trigger methods
  • const form = documnet.forms[0] will allow selection of a single form from a page of multiople forms
  • button with the type set to 'reset' will reset the form.
  • Disabled in the button with type as submit will block the submit button until Disabled is false.

Chapter 12 Object Oriented Programming

  • Static Methods: Static can be used infront of a function. It is called just by just using the class rather than the instances of the class
  • Prototype allows us to create new properties and methods even after the instanciation of the class.
  • To find the prototypes, object.getPrototypeOf(instance) or instance.__proto__
  • Object.assign(objecta, objectb) will combine two objects into one.
